## Project Overview
NJ Studio is a professional photography company based in South Africa, passionate about capturing life's precious moments to create unforgettable memories. This static HTML website showcases services, portfolio, pricing, and contact options. Built as a multi-page site with responsive design elements.

## Website Goals and Objectives
- Showcase photography services and previous work to attract clients.
- Provide clear pricing for various photoshoot packages (photos & videos).
- Enable easy contact via form and details.
- Highlight photographer's background and passion.
- Serve as a portfolio for student/academic project.

## Key Features and Functionality
- **Navigation**: Consistent menu across pages (Home, About, Contacts, Gallery, Pricing).
- **Home Page**: Hero section introducing NJ Studio.
- **About Page**: Photographer bio, experience, education.
- **Pricing Page**: Packages for Wedding (R500-2700), Maternity (R300-800), Graduation (R300-1000), Outdoor (R500-1700), Birthday (R450-1000).
- **Contacts Page**: Inquiry form + email (NJSTUDIO@gmail.com), phone (0812746784).
- **Gallery Page**: Responsive grid of previous work samples.
- Pure HTML with external images from Bing.
