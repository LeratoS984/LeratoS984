- `index.html` (home)
- `about.html` (about)
- `contacts.html` (contacts)
- `gallery.html` (previous work)
- `pricing.html` (photoshoot prices)

## Styling
All styling lives in `style.css`.

## Updates made
### Gallery alignment (CSS)
To make the gallery images align into consistent rows, these rules were added/updated:
- .gallery-grid` is centered and constrained for better row stability
- .gallery-grid .responsive` uses predictable box sizing and alignment (`inline-block` grid behavior)

Result: no more blank/extra tile slots, and the pictures line up consistently across screen sizes.
