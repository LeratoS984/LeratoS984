 NJ Studio — Photography Website

NJ Studio is a professional photography company based in South Africa, passionate about capturing life’s precious moments and turning them into unforgettable memories. This project is a static HTML website built as a student portfolio and academic submission, designed to showcase services, portfolio, pricing, and contact options in a clean, modern, and responsive way.



Project Overview
The website serves as both a professional portfolio and an academic project. It highlights NJ Studio’s photography services, the photographer’s background, and examples of previous work. Built with pure HTML, CSS, and JavaScript, it demonstrates practical web development skills without relying on external frameworks.


 Goals and Objectives
- Present NJ Studio’s photography services in a professional, appealing way.  
- Showcase previous work through a responsive gallery.  
- Provide clear pricing for different packages (photos and videos).  
- Make it easy for clients to get in touch via a contact form and direct details.  
- Share the photographer’s background, passion, and experience.  
- Function as a portfolio piece for academic evaluation.  


 Key Features and Functionality
- Navigation: Consistent menu across all pages (Home, About, Contacts, Gallery, Pricing).  
- Home Page: Hero section introducing NJ Studio with a call-to-action button.  
- About Page: Photographer bio, education, and experience.  
- Pricing Page: Transparent packages for weddings (R500–2700), maternity (R300–800), graduation (R300–1000), outdoor shoots (R500–1700), and birthdays (R450–1000).  
- Contacts Page: Inquiry form with validation, plus direct email (NJSTUDIO@gmail.com) and phone (0812746784).  
- Gallery Page: Responsive grid of previous work samples with lightbox preview.  
- Technical Enhancements:  
  - Toast notifications for form feedback.  
  - Active navigation highlighting.  
  - Mobile menu toggle with accessibility features.  
  - Lightbox with zoom, slideshow background, and keyboard navigation.  



 Styling (style.css)
- Custom color palette with accents and muted tones.  
- Responsive grid layouts for sections, gallery, and pricing.  
- Sticky navigation bar with hover and active states.  
- Mobile-friendly menu toggle.  
- Smooth transitions and hover effects.  
- Toast notifications styled for clarity.  
- Lightbox design with zoom controls and slideshow backdrop.  


 JavaScript Features (script.js)
- Toast Notifications: Feedback for form validation and user actions.  
- Active Navigation: Highlights the current page in the navbar.  
- Mobile Menu Toggle: Accessible hamburger menu with ARIA attributes.  
- Home CTA Button: Redirects “Work with me” button to the contacts page.  
- Contact Form Validation: Ensures all fields are filled, validates email format, and provides clear feedback.  
- Gallery Lightbox:  
  - Click or keyboard-accessible image preview.  
  - Zoom in/out with buttons, mouse wheel, or click toggle.  
  - Previous/Next navigation with arrows or keyboard keys.  
  - Background slideshow effect for immersive viewing.  
  - Accessible with ARIA roles and focus management.  


 Responsiveness
- Breakpoints at 920px and 620px for mobile and tablet layouts.  
- Navigation adapts to a mobile menu.  
- Gallery and pricing grids adjust from four columns to two, then one.  
- Images resize while preserving aspect ratios.
